# EntentPay Landing Page

A responsive landing page for EntentPay — a fast and secure payment solution for online and offline businesses.

---

## 🚀 Features

* Responsive layout (mobile / desktop)
* Clean semantic HTML
* BEM methodology
* Optimized assets (images, CSS, JS)
* Simple build process without heavy bundlers

---

## 📁 Project Structure

```
project/
  assets/
    fonts/
    img/
    lottie/
  css/
    style.css
  js/
    main.js
  index.html
  dist/        # generated after build
```

---

## 🚀 Development

### Live Server (VS Code)

1. Install **Live Server** extension
2. Open `index.html`
3. Right-click → **Open with Live Server**

👉 The page will automatically reload on changes

---

## ⚙️ Build

Project uses a lightweight build setup (no bundlers).

### Run build

```
npm run build
```

---

## 📦 Output

After build, a `dist/` folder will be generated:

```
dist/
  index.html
  css/style.css
  js/main.js
  assets/
```

👉 All files are:

* minified
* optimized
* ready for production
